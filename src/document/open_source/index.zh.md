## 如何使用

- 你可以在个人或商业项目中使用本仓库代码，但需遵循 MIT 许可证关于署名与许可条款的要求。
- 若某文件包含不同的许可声明，请以该文件声明为准。

## 贡献

欢迎任何形式的贡献。贡献流程建议：

1. Fork 本仓库。
2. 新建 feature 分支。
3. 提交修改并添加必要的测试。
4. 发起 Pull Request，并写明修改说明。

请遵守仓库内的行为准则（Code of Conduct）与贡献指南（CONTRIBUTING.md）。
